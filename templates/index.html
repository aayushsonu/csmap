CSMAP Stands for:

  • C : Caeser Cipher
  • S : Strong Password Generator
  • M : MAC Spoofer
  • A : ARP Spoofer
  • P : Port Scanner

CSMAP Help

sudo python3 csmap.py -h
sudo python3 csmap.py --help

CSMAP Help In Web Page

sudo python3 csmap.py help web

Caeser Cipher Tool

Encryption:

sudo python3 csmap.py -c <text> -k <key>

Decryption:

sudo python3 csmap.py -d <text> -k <key>

Random Password Generator

sudo python3 csmap.py -l <length of password>

MAC Spoofer

sudo python3 csmap.py -m

ARP Spoofer

sudo python3 csmap.py -a <router IP> -v <victim IP>

Port Scanner

sudo python3 csmap.py -p <port number> -v <IP Address>
